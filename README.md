# JavaBot AI
### AI-Powered Java Learning Assistant

JavaBot AI is an intelligent web-based chatbot designed to help users learn Java programming through real-time AI responses. It uses Google Gemini API to generate accurate explanations, code solutions, and debugging help, making Java learning easier and more interactive.

---

## Features
- Real-time AI chat interface  
- Java-focused intelligent responses  
- Code generation and explanation  
- Debugging support for Java programs  
- Beginner-friendly learning assistant  
- Responsive UI design  

---

## Tech Stack
- HTML  
- CSS  
- JavaScript  
- Google Gemini API  
- Fetch API / Async-Await  
- VS Code  

---

## How It Works
1. User types a Java-related question  
2. Input is sent to Gemini AI via API request  
3. AI processes the prompt as a Java tutor  
4. Response is displayed in chat interface
